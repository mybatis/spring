MyBatis-Spring 어댑터는 MyBatis sql 매핑 프레임워크에 사용하기 쉬운 Spring bridge 입니다.

지원되는 버전
------------------

- 1.3.x - Java 6 및 7에 대한 지속적인 지원
- master (2.0.x) - Java 8, Spring 5 및 Junit 5 이상 기타 Java 8 요구사항
